String Interpolation Examples

Things We can do using String Interpolation

  • 4+5 : {{4+5}}
  • Num1 : {{num1}}
  • Num1+Num2 : {{num1+num2}}
  • Full Name : {{fName+" "+lName}}
  • Terms and Conditions : {{termsAndConditions}}
  • Person : {{person1.Name + " is from "+person1.Address}}
  • Full Name : {{getFullName()}}

Things We cannot do using String Interpolation

  • Cannot create new Objects in SI expression
  • Cannot write chained expression
  • Cannot write comma separated expression
